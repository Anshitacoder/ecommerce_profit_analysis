# E-commerce Profitability Analysis

## Problem Statement

The company is experiencing increasing sales but declining profitability.
This project analyzes the root causes behind this issue using real-world e-commerce data.

## Dataset

* Olist Brazilian E-commerce Dataset
* Includes orders, customers, payments, products, and shipping data

## Tools & Technologies

* Python (Pandas, NumPy)
* Matplotlib (Visualization)
* Jupyter Notebook(For Analysis)

## Analysis Performed

### Revenue vs Profit Trend

* Revenue shows an increasing trend
* Profit is not growing proportionally

### Top Loss-Making Categories

* Certain product categories consistently generate losses
* Indicates pricing or cost inefficiencies

### Profit Distribution

* Most orders are profitable
* A small percentage of loss-making orders impacts overall profit

## Key Insights

* High shipping costs significantly reduce profit
* Some categories are inherently unprofitable
* Loss-making orders, though few, have a high impact

## Recommendations

* Optimize shipping/logistics cost
* Re-evaluate pricing strategy
* Avoid selling products where cost exceeds revenue

## Conclusion

Despite strong sales growth, inefficient cost management—especially shipping—leads to declining profitability.
